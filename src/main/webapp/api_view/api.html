Bienvenido a la API Rest de Enhanced Universe

Para acceder a la API de galaxias: (Soporta GET, POST, PUT, DELETE)


/enhanced_universe/rest/galaxia

Para acceder a una galaxia específica: /enhanced_universe/rest/galaxia/"nombre de la galaxia"

Ejemplo: /enhanced_universe/rest/galaxia/via lactea

Respuesta: { "id": 1, "nombre": "via lactea", "tipogalaxia": { "id": 1, "nombre": "espiral" }, "alto": null, "ancho": null, "profundidad": null, "diametro": 100000, "distanciatierra": 0 }

Para ingresar: /enhanced_universe/rest/galaxia - enviar un json con la informacion de la galaxia a ingresar

Para eliminar: /enhanced_universe/rest/galaxia/"nombre de la galaxia"

para actualizar: /enhanced_universe/rest/galaxia - enviar un json con la información de la galaxia a editar
